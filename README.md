# Apple Inc. Dynamic 3-Statement Financial Model & Scenario Forecaster

## Project Overview
This repository contains a fully integrated, dynamic 3-statement financial model for Apple Inc., built entirely in Excel. The model maps historical financial performance and projects a 5-year forecast (2026E–2030E) using automated historical baselines and dynamic macroeconomic assumptions.

## Data Attribution & Sourcing
* **Historical Data:** All historical financial data (2023–2025) was sourced directly from Apple Inc.'s publicly available SEC 10-K filings. 
* **Forecast Data:** The forward-looking projections (2026E–2030E) are educational assumptions engineered for this model to demonstrate corporate finance forecasting techniques.

## Core Architecture & Features
* **Integrated Financial Statements:** A fully linked Income Statement, Balance Sheet, and Cash Flow Statement. The architecture employs strict fixed asset roll-forwards (PP&E, CapEx, Depreciation) and working capital turnover metrics to achieve a self-balancing ecosystem (Assets = Liabilities + Equity) with a $0 discrepancy reconciliation.
* **Master Scenario Toggle:** An interactive "Assumptions Dashboard" utilizing `CHOOSE` functions to allow users to seamlessly switch the entire 5-year forecast between **Base**, **Bull**, and **Bear** growth scenarios in real-time.
* **Executive Summary Dashboard:** A presentation-ready front-end featuring stripped gridlines and responsive charting that synthesizes top-line growth trends, net income margins, and cash balances for C-suite stakeholders.

## Technical Skills Demonstrated
* Corporate Finance & FP&A Modeling
* Advanced Excel (Scenario Analysis, Dynamic Linking, Logical Functions)
* Accounting Logic & Reconciliation
* Financial Data Visualization 

## How to Use This Model
1. Download the [Dynamic Budget Variance & Cash Flow Forecaster.xlsx](Dynamic%20Budget%20Variance%20%26%20Cash%20Flow%20Forecaster.xlsx) file.
2. Open the **Executive Summary** tab.
3. Change the **Active Scenario** toggle (1 = Base, 2 = Bull, 3 = Bear) to watch the charts, revenues, and bottom-line margins automatically recalculate across the entire ecosystem.
